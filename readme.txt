En este programa el usuario puede ver el tipo de cambio del dolar a pesos peruanos
puede ver la tasa de venta y la tasa de compra

El usuario puede ingresar una cantidad y presionando el boton dolares a pesos se hara la conversión
de utilizando el precio de venta.
Si el usuario presiona el boton soles a dolares se hará la conversión utilizando el precio de compra.

Nota: Utilicé la moneda peruana porque no encontre un API con quetzales.
